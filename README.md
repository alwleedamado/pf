# pf

[![CI](https://github.com/alwleedamado/pf/actions/workflows/release.yml/badge.svg)](https://github.com/alwleedamado/pf/actions)
[![Go Reference](https://pkg.go.dev/badge/github.com/alwleedamado/pf.svg)](https://pkg.go.dev/github.com/alwleedamado/pf)

Clean SDK & build caches in one command.

## Install
```bash
go install github.com/alwleedamado/pf@latest

| Command          | Description                  |
| ---------------- | ---------------------------- |
| `pf list`        | List global & project caches |
| `pf interactive` | Expandable TUI               |
| `pf clean <dir>` | Delete with `--dry-run`      |
